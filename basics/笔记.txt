一： 变量的知识要点
    1. 利用 var来声明变量，变量类型写在变量名后
    2. 编译器可以推测变量类型
    3. 没有char，只有rune
    4. 原生支持复数类型

二： 常量的知识要点
    1. 利用 const 声明常量，声明的常量是固定不变的，在程序运行期间不能被修改。
    2. 常量可以放在函数外面，供所有包调用，放在函数里，供当前函数调用。
    3. iota 是枚举型常量，在 const 声明后，iota 的值都会递增。多个常量被定义在同一行，则共享一个 iota 值。
       iota 出现在枚举常量的表达式中，他的值为表达式的值。
    4. 常量和变量都可以用 := 来赋值。

三： 条件语句
    1. if 语句
        ①. 根据某个条件的结果来决定是否执行
        ②. 可选 else 语句，在结果为 false 时执行另一个代码
        ③. 多种条件使用可使用嵌套的if语句或者使用 else if 语句。
        ④. 可使用段变量声明语法来定义一个变量，变量的作用区域仅限于 if 语句中。
    2. switch 语句
        ①. 替代多个 if else 语句
        ②. case 是switch语句中的关键字，当 switch 匹配到某个 cash 表达式，该 cash 语句块会被执行。
        ③. 可使用多个 cash 如果值等于多个值中的其中一个，则代码块会被执行。

四： 函数
    1. 返回值类型写在最后面
       ①: func function_name(parameter_list) (return_type) {
             // 函数体
          }
    2. 可返回多个值
    3. 函数可以作为参数传递给其他参数，也可以将函数作为返回值返回。
    4. 没有默认参数，可选参数。
    5: 函数名以字母或下划线开始。
    6：函数可以又可变数量的参数。可以用语法 '...' 表示可变参数。

五： 指针
    1. 指针是一种变量类型，存储了一个内存地址，用 * 来表示
    2. 可以存储、传递、修改、引用
    3. 用 & 来获取指针地址。
    4. 使用前判断指针地址是否为空，指向类型是否分配。
